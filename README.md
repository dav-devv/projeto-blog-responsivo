The Dev News - Blog Responsivo
🎓 Projeto de Estudos - Escola DNC
Este projeto foi desenvolvido como parte dos estudos práticos em desenvolvimento web na Escola DNC, com o objetivo de aplicar conceitos fundamentais de Front-end.

📝 Sobre o Projeto
O The Dev News é uma página web estática que simula a interface de um blog de tecnologia e desenvolvimento. O foco principal deste projeto foi construir um layout moderno, atraente e, acima de tudo, 100% responsivo, garantindo uma boa experiência de leitura tanto em monitores ultrawide quanto em telas de celulares.

🚀 Tecnologias e Conceitos Utilizados
Neste projeto, foram aplicadas as seguintes tecnologias e técnicas:

HTML5: Estruturação semântica da página (<header>, <main>, <section>, <aside>, <footer>).

CSS3: Estilização visual e tipografia (Google Fonts - Montserrat).

CSS Grid Layout: Criação da estrutura principal de colunas e organização da galeria de cards de posts.

Flexbox: Alinhamento de elementos internos e centralização.

Media Queries: Adaptação do layout para dispositivos móveis, alterando o número de colunas e ajustando tamanhos de fontes e contêineres.

Posicionamento CSS: Uso de position: relative e absolute para sobreposição elegante de textos em imagens de destaque com gradientes.

📱 Funcionalidades de Destaque
Design Responsivo: O layout se adapta quebrando as 3 colunas originais para uma única coluna em telas menores que 767px.

Imagens Proporcionais: Uso de aspect-ratio e object-fit para garantir que as capas dos artigos não fiquem distorcidas em diferentes tamanhos de tela.

Footer Dinâmico: O rodapé se mantém no fundo da página independentemente da quantidade de conteúdo na tela.
